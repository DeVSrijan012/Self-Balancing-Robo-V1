# Self-Balancing-Robo-V1
A PID-based self-balancing robot using Arduino Uno, MPU6050, and an L298N motor driver to achieve real-time autonomous balancing.

## 📂 Project Structure

```text
Self-Balancing-Robo-V1/

│
├── 📁 PID_TUNINING_INSTRUCTIONS/
```
